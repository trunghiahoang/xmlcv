# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-XX-XX

### Added
- Initial release
- Dynamic XML element processing
- Support for various XML structures
- HTML output with customizable CSS
- PDF export support (WeasyPrint/pdfkit)
- CLI interface (`xmlcv`)
- Python API for programmatic use
- XML structure analysis
- Custom element processor registration
- Table of Contents generation
- Navigation sidebar
- Support for various XML elements:
  - Document, Title, Statement
  - Chapter, Section, Subsection, Article
  - Paragraph, Sentence, Item, Subitem1, Subitem2
  - TableStruct, Table, TableRow, TableColumn
  - Provision, AppdxTable, AppdxStyle, AppdxNote
  - Ruby annotations, Sub (subscript)
  - List, Note, QuoteStruct, Style, Fig, ArithFormula

### Features
- Automatic XML schema analysis
- Extensible processor system
- Configurable output options
- Batch directory conversion
- Index page generation
